What is this?

You can think of this website as an archive of all of all of my web 3D projects.

The biggest advantage of web based graphics frameworks over the more traditional ones (DirectX, Vulkan, Metal) is that your projects can easily be shared to any device. With that in mind, I thought it'd make sense to create a website to share my 3D programs with others more easily.

Every project here is open source. The source code for this website and all projects inside it can be found on my Github.
